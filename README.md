Overview
--------
The Student Progress Tracker System is a simple Python-based mini project designed for beginners. 
It helps in understanding core Python concepts by building a real-life application that collects 
student details, calculates marks, and generates a progress report.

This project is ideal for B.Tech / Diploma students who are learning Python fundamentals.

Purpose of the Project
----------------------
The main goal of this project is to:
- Practice basic Python programming concepts
- Learn how to take user input and process data
- Understand conditions, loops, and functions
- Generate a clean and readable output report

What This Program Does
----------------------
1. Displays a welcome screen
2. Asks for student details (Name, Class, Roll Number)
3. Takes marks for multiple subjects
4. Calculates:
   - Total marks
   - Average marks
   - Percentage
   - Grade
   - Pass/Fail result
5. Shows highest and lowest marks
6. Displays remarks based on performance
7. Allows the user to re-enter marks or view the report again

Python Concepts Used
--------------------
- Variables and Data Types
- User Input and Output
- Conditional Statements (if, elif, else)
- Loops (for, while)
- Functions
- Lists
- String formatting

How to Run the Project
---------------------
1. Make sure Python is installed on your system
2. Save the program file as student_progress_tracker.py
3. Open terminal or command prompt
4. Run the command:
   python student_progress_tracker.py
5. Follow the on-screen instructions

Output
------
The program generates a neatly formatted student progress report showing all calculated results 
in a clear and understandable format.

Conclusion
----------
This mini project is a beginner-friendly application that connects theory with practice. 
By completing this project, students gain confidence in Python programming and understand 
how basic concepts are used in real-world scenarios.
